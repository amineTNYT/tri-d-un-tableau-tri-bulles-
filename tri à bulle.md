# tri-d-un-tableau-tri-bulles-
Ce programme Python implémente un algorithme de tri de tableau en utilisant la méthode du tri a bulles(sort). Le programme permet à l'utilisateur de :  Saisir la taille d'un tableau . Remplir le tableau avec des valeurs entières  Afficher le tableau avant le tri  Trier le tableau en ordre croissant  Afficher le tableau après le tri
